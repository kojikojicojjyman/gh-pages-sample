重力場のある座標系での一次元格子を考える。質量 m の各質点の変位を q n とし、バネの伸びを

(1) r n = q n − q n − 1 Δ x

とする。また質点間の平衡距離を Δ x = 1 とする。線密度を ρ とすれば m = ρ Δ x = ρ である。

バネのポテンシャルが

(2) ϕ ( r ) = k ( C r + 1 2 r 2 + 1 2 r 3 + 1 8 r 4 )

のとき、この質点系の運動方程式は

(3) k f ( r ) = − ϕ ′ ( r ) = − k ( C + r + 3 2 r 2 + 1 2 r 3 )

として、

(4) ρ q n ¨ = k Δ x f ( q n − q n − 1 Δ x ) − k Δ x f ( q n + 1 − q n Δ x ) − ρ g

より

(5) m q n ¨ = k f ( q n − q n − 1 ) − k f ( q n + 1 − q n ) − m g

となっている。ここで、変位 q n ( t ) = y ( x , t ) = y ( n Δ x , t ) として滑らかな関数 y ( x , t ) を導入して、物体内部の座標で x となる質点の、空間上の位置を z ( x , t ) = x + y ( x , t ) とおく。

少し話は戻るが、各質点の位置を

(6) Q n ( t ) = z ( x , t ) = z ( n Δ x , t ) = n Δ x + y ( Δ x , t )

と置いて、ポテンシャルを書き直してみると

(7) Q n − Q n − 1 = Δ x + q n − q n − 1

で

f ( r ) = − C − r − 3 2 r 2 − 1 2 r 3 = − C − 1 2 ( r + 3 r 2 + r 3 ) = − C − 1 2 ( 1 + − 2 − 2 r + 1 + 3 r + 3 r 2 + r 3 )

より、

(8) f ( r ) = − C ′ + 2 ( 1 + r ) − 1 2 ( 1 + r ) 3

なので、

(9) R n = Q n − Q n − 1 Δ x (10) F ( R ) = − C ′ + 2 R − R 3 2

とすると、 R n = 1 + r n だから

(11) F ( R n ) = f ( r n )

という関係になる。また Q n ˙ = q n ˙ なので、 Q n についての運動方程式が

(12) ρ Q n ¨ = k Δ x F ( Q n − Q n − 1 Δ x ) − k Δ x F ( Q n + 1 − Q n Δ x ) − ρ g

つまり

(13) m Q n ¨ = k F ( Q n − Q n − 1 ) − k F ( Q n + 1 − Q n ) − m g

と得られる。ここで F ( R ) は、仮に任意定数を C ′ = 0 として

(14) F ( R ) = 2 R − R 3 2

で与えられる。この系のポテンシャルは、任意定数を 0 とすれば

(15) Φ ( R ) = − k R 2 + k R 4 8

となっている。

(16) ϕ ( r ) = k 2 r 2 + k ′ 4 r 4

の形になっているものは、Fermi-Pasta-Uram格子 β モデルと呼ばれており、よく研究されている。
